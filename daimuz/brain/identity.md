# 🧬 Identidad — Lopbuk

> ¿Qué es Lopbuk exactamente?

## Definición

**Lopbuk** es una plataforma SaaS multi-tenant para operar negocios. No es un simple POS ni un simple inventario — es un **sistema operativo de negocio** que unifica en una sola interfaz:

- Inventario y stock en tiempo real
- Punto de venta (POS)
- Gestión de pedidos y delivery
- CRM de clientes
- Finanzas y flujo de caja
- Operaciones gastrobar (mesas, recetas, merma)
- Presencia digital (tienda online, menú, portafolio)
- Reportes y analytics

## Audiencia

| Tipo de negocio | Módulos clave |
|---|---|
| Gastrobar / Restaurante | restbar · recipes · merma · gastrobar-ops · pos |
| Retail / Tienda | inventory · pos · sales · storefront |
| Ecommerce | storefront · orders · delivery |
| Servicios | services · workorders · portfolio |
| Inmobiliaria | realestate |

## Propuesta de valor core

> **"Control total en tiempo real, sin complejidad."**

1. Un solo sistema para todo el negocio
2. Interfaces por rol (cada empleado ve solo lo que necesita)
3. Datos en tiempo real para decidir rápido
4. Escala con el negocio (multi-sede, multi-rol, multi-módulo)

## ADN Técnico

- **Multi-tenant** → un sistema, múltiples negocios
- **Modular** → cada negocio activa solo lo que usa
- **Tiempo real** → Socket.io para eventos críticos
- **Mobile-first** → paneles optimizados para tablets/móvil

## Lo que NO es Lopbuk

- No es un ERP pesado tipo SAP
- No requiere instalación local
- No tiene curva de aprendizaje larga
- No obliga a usar todos los módulos
