Coloca aquí el sonido de notificación del widget de WhatsApp:

  messenger-notification.mp3

El componente WhatsAppFloatingWidget lo reproduce al mostrar el mensaje
(/sounds/messenger-notification.mp3 por defecto).

Si el archivo no existe, el widget sigue funcionando sin sonido (se ignora
silenciosamente). Puedes cambiar la ruta con la prop soundUrl.

Sugerencia: usa un MP3 corto (~1s), libre de derechos, tipo "pop"/Messenger.
